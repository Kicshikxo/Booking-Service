## О проекте

Booking Service - это веб-приложение для бронирования мест на мероприятия.  
Система позволяет пользователям регистрироваться, создавать события и бронировать доступные места.  
Каждый пользователь может забронировать место на событие только один раз.

## Попробовать прямо сейчас

Используйте Booking Service прямо в браузере:

- [booking.kicshikxo.ru](https://booking.kicshikxo.ru)

## Установка и запуск

### Клонирование репозитория

```bash
git clone https://github.com/Kicshikxo/Booking-Service.git
cd Booking-Service
```

### Установка зависимостей

```bash
npm install
```

### Запуск в режиме разработки

```bash
npm run dev
```

### Сборка релизной версии

```bash
npm run build
npm run preview
```
